# 🧮 Nisar's Scientific Calculator

**Professional Streamlit calculator with 14 advanced functions** including trigonometry, logarithms, power functions, and complete history tracking.

## ✨ **Features**
- **Basic**: Addition (+), Subtraction (-), Multiplication (×), Division (÷)
- **Advanced**: Modulus (%), Square (x²), Square Root (√x), Inverse (1/x)  
- **Scientific**: sin, cos, tan, log, e^x, Power (x^y)
- **History**: Track last 10 calculations
- **Responsive**: Professional Times New Roman UI design

## 🚀 **Quick Start**

```bash
# Clone repository
git clone https://github.com/nisar-ai/scientific-calculator.git
cd scientific-calculator

# Install dependencies
pip install -r requirements.txt

# Run app
streamlit run app.py
